# MyTrips — Vibey Earth Map

A lightweight starter website for your travel memories.

## Features
- Interactive map with clickable memory pins.
- Add memories with date/time, coordinates, photo URL or local photo upload, and vibe note.
- Place autocomplete via OpenStreetMap Nominatim (fills lat/lng automatically).
- Timeline cards sorted from newest to oldest.
- Data persisted with browser localStorage.

## Run locally
```bash
python3 -m http.server 4173
```
Open `http://localhost:4173` in your browser.
